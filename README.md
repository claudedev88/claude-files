# claude-files

A personal collection of Claude configuration and context files (`CLAUDE.md` files and related notes), kept public so they're easy to grab from any computer.

## What's here

The `agentforce-project-starter/` folder contains the `CLAUDE.md` and notes for an Agentforce Project Starter — a Claude Code setup assistant for Salesforce Agentforce development.

## Getting these files on another computer

**Grab a single file** — open it on GitHub, click **Raw**, then save the page.

**Clone the whole repo:**

```bash
git clone https://github.com/claudedev88/claude-files.git
```

**Pull the latest updates later:**

```bash
cd claude-files
git pull
```

## A note on privacy

This is a **public** repository — anything committed here is visible to everyone, and stays in the Git history even after deletion. Don't commit API keys, tokens, or private file paths. The included `.gitignore` blocks common secret files as a safety net.
